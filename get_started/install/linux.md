## Requirements

- Hypervisor: at least one of
  - [Linux] QEMU 2.0 or later
  - [Linux] Xen 4.5 or later (for Xen support)

## Install

[the current version](../../release_notes/latest.md) supports the following Linux distros:

- Ubuntu ( > 14.04 )
- CentOS/RHEL 7.x
- Fedora ( > 20 )
- Debian ( > 7.0 )
